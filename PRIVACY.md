# Privacy Policy for RenForge

**Effective Date:** 2025-01-01

RenForge is a free and open-source desktop application provided “as is” for local file renaming operations.

---

## 1. Data Collection

RenForge **does not collect, store, transmit, or process any personal data**.

- No user accounts
- No sign-in or authentication
- No analytics
- No telemetry
- No cookies
- No background services

The application runs entirely **offline** on the user’s local machine.

---

## 2. Network Communication

RenForge **does not communicate with any external servers**, APIs, or cloud services.  
All operations are performed locally on files selected by the user.

---

## 3. File Access

RenForge only accesses files and folders that the user explicitly selects.  
No file data is transmitted outside the user’s system.

---

## 4. Third-Party Services

RenForge does not integrate with or depend on any third-party online services.

---

## 5. Responsibility and Usage

RenForge is a utility tool intended for technically aware users.

- Users are solely responsible for how they use the application.
- RenForge does not guarantee the safety, correctness, or suitability of file operations.
- Users are encouraged to review previews carefully before applying changes.

Use of the application implies acceptance that **the authors take no responsibility for data loss, misuse, or unintended outcomes**.

---

## 6. Open Source Transparency

RenForge is open source.  
The full source code is available for inspection at:

https://github.com/blackarck/renforge

Users are encouraged to review, modify, or build the application themselves if desired.

---

## 7. Support and Contact

RenForge is provided with **limited support**.

For issues, questions, or discussions, please use the GitHub repository:

https://github.com/blackarck/renforge
